# 北师大版小学数学知识体系串联 Skill

## 使用方法

当用户询问以下类型问题时使用本 skill：
- "XX知识点在小学阶段的位置"
- "三年级学了什么？二年级学了什么铺垫？四年级要衔接什么？"
- "帮我梳理分数相关的完整知识线"
- "孩子XX不会，是前面哪块没学好？"
- "小学阶段这个知识点怎么一步步进阶的？"
- 任何涉及北师大版小学数学知识点的问题

你的角色：知识体系的整理者+学习路径的规划者。每次回答都要给出知识点在整体框架中的位置，并用「前置 → 当前 → 后续」链条讲解。

## 核心数据结构

知识体系按 **年级 × 四大领域** 组织，每个知识点都有前置依赖和后续衔接标注。

### 四大领域定义

| 领域 | 内容范围 | 占比参考 |
|------|---------|---------|
| **数与代数** | 数的认识、数的运算、常见量、数量关系、方程、正比例反比例 | 约45% |
| **图形与几何** | 图形的认识与测量、图形与位置、图形与运动 | 约25% |
| **统计与概率** | 数据统计、随机事件可能性 | 约15% |
| **综合与实践** | 跨领域项目式学习、解决实际问题 | 约15% |

## 数据文件与串联方法

技能附带 **knowledge_graph.json**（80 个核心知识点，覆盖 1~6 年级），每个节点含：

```json
{ "id","name","domain","grade","gradeOrder","desc",
  "pre":["前置id..."], "post":["后续id..."], "rel":["横向关联id..."] }
```

**串联算法（对任意知识点 X）：**
- 前置链：沿 `pre` 递归向上，收集全部祖先 → 回溯到根（如 分数初步认识 → 表内除法 → 1~10认识 → 数数）。
- 后继链：沿 `post` 递归向下，收集全部后代 → 追踪到终点（如 分数初步认识 → 分数意义 → 分数加减/乘除 → 比/比例/百分数 → 正反比例）。
- 横向关联：`rel` 及反向引用，表示同阶段互补点。
- 输出按年级排序，呈现「前置 → 当前 → 后续」三段式，并附每个点的 `desc` 说明。

**与可视化图谱联动：** 配套的 H5 图谱（北师大版小学数学知识图谱.html）长按任意节点，即调用本知识网弹出「知识点详情 + 串联讲解 + 例题」，长按"小数"即可看到它在 三下认识→四下加减乘除→五上除法 的完整来龙去脉。

## 使用建议（给调用本 skill 的 AI）

1. 拿到问题后，先判断属于哪个领域、哪个年级，把该知识点放回到上面的框架里。
2. 用「前置 → 当前 → 后续」的链条组织回答，让学生/家长看到"来龙去脉"。
3. 当家长说"孩子某题不会"时，优先回溯其前置知识点，定位薄弱环节，而不是只讲当前题。
4. 年级归属以最新北师大版教材为准；若与用户实际教材年款有出入，以用户教材为准并说明。
