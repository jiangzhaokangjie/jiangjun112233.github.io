# jiangjun112233.github.io

本仓库托管 WorkBuddy / CodeBuddy 技能：**北师大版小学数学知识图谱**。

## 技能：beishida-math-knowledge

把北师大版小学（1~6 年级）数学零散知识点建成网状图谱，从任一知识点出发可回溯前置、追踪后继、串联整个小学阶段。涵盖数与代数、图形与几何、统计与概率、综合与实践四大领域，含 80 个知识点与 1~6 年级完整大纲。

## 安装（一行命令）

```bash
npx skills add jiangzhaokangjie/jiangjun112233.github.io -g -y
```

装好后，在 WorkBuddy 对话里输入 `@skill:beishida-math-knowledge`，或直接说：
"把 小数、分数、百分数 串联讲解一下"。

## 文件说明

- `SKILL.md`：技能说明 + 1~6 年级大纲（带前置/后续）+ 串联讲解方法
- `knowledge_graph.json`：80 个知识点的网状关系图（机器可读）
- `安装说明.txt`：不会命令行时的手动安装步骤

详见压缩包内 `安装说明.txt`。
